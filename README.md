# Digital Workload Management System (DWMS)

Developer: Panya Jeerachat  
Faculty of Engineering  
Mahasarakham University

## Description

Digital Workload Management System (DWMS) is a digital system developed for managing audiovisual academic workloads and services.

The system supports:

- Workload recording
- Service tracking
- Digital documentation
- Dashboard reporting
